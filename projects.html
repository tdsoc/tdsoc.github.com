How to contribute

All the code and the documentation of TDSoC will be freely available on the TDSoC website and can be checked out and reused for free (see the license). The establishment of the new projects follows the guide lines of the TDSoC policies. Every existing project has a project leader, whom you should contact if you want to partecipate. Any software contribution by partecipant will be reviewed by the project leader before becoming official part of the TDSoC project. The roles inside each project are agreed between the project leader and the partecipants.
